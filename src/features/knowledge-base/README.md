# Knowledge Base Module

## Purpose
Documentation repository for storing and managing knowledge base documents with markdown support.

## Status
Under construction - being migrated from existing codebase.

## Structure
- `components/` - UI components and page components
- `hooks/` - Custom React hooks for data fetching
- `queries.ts` - Amplify GraphQL queries
- `mutations.ts` - Amplify GraphQL mutations
- `types.ts` - TypeScript type definitions
- `utils.ts` - Helper functions (markdown processing)
